# Boilerworks Rails + Hotwire

> Full-stack Rails 8 with Hotwire (Turbo + Stimulus) for marketplace, social, and content apps.

**Status:** Building

## Stack

| Layer | Technology |
|-------|-----------|
| Backend | Rails 8 |
| Frontend | Hotwire (Turbo + Stimulus) |
| Templates | ERB |
| Database | Postgres 16 |
| Cache | Redis 7 (Solid Cache) |
| Task Queue | Solid Queue |
| CSS | Tailwind CSS |
| Asset Pipeline | Propshaft |

## Features

- Rails 8 with Hotwire for SPA-like interactivity without a JS framework
- Turbo Drive, Frames, and Streams for partial page updates
- Stimulus controllers for client-side behavior
- Solid Queue for background jobs
- Solid Cache + Redis for caching
- Session-based auth with Devise and group-based RBAC
- Tailwind CSS with Propshaft
- Docker Compose dev environment (Postgres, Redis, Mailpit)
- Comprehensive test suite with Minitest
